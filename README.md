# 🎮 Online Gaming Leaderboard System

## 🚀 Overview
This project is an end-to-end data engineering pipeline that simulates gaming data and generates a real-time leaderboard.

## 🧠 Architecture
Data Generation → Ingestion → Processing → Database → Dashboard  
                 ↑  
              Airflow (Automation)

## ⚙️ Tech Stack
- Python
- Pandas
- SQLite
- PySpark
- Apache Airflow
- Streamlit

## 🔄 Features
- Automated pipeline using Airflow
- 200+ synthetic dataset generation
- Near real-time updates (every 2 minutes)
- Leaderboard ranking system
- Interactive dashboard (Top 3 + charts)
